Visual Grammar

黑帮张力不是滤镜

一张真正有用的参考图，至少要回答四件事：谁掌控场面、谁正在承受压力、权力靠什么物件被看见、缩成英雄卡面后还剩下什么。

01
↔

把对峙放在动作前

两张脸、极短距离、无人先动。冲突已经发生，但暴力仍被克制，观众会主动补完后果。

适合：毒枭会谈 / 黑手党审判
02
△

让权力形成三角

老板居中，家族与执行者分列两侧；所有视线最终回到核心人物，服装只负责标记层级。

适合：家族群像 / 组织关系图
03
□

留白比金链更昂贵

大佬不需要堆满装备。空房间、单把椅子、一块表，让“其他人已经被清场”成为价值感。

适合：SSR 立绘 / 招募页
04
✕

伤痕必须有代价

血、旧伤、破损西装都应指向刚结束的决定。角色不是脏，而是承受过权力的反噬。

适合：觉醒皮肤 / 剧情节点
Featured Translation · MobLand

家族不是背景，
是大佬身上的第二套武器。

《MobLand》的价值不在高级西装本身，而在西装、室内陈设和人物站位共同说明：这不是临时犯罪团伙，而是一套已经运转多年的继承制度。

肩线稳定、衣长压低重心；不靠夸张肌肉。
家族戒、老照片、刻名打火机、私人会所钥匙。

Faction Translation

八阵营怎么用

参考不是照抄服装。下面把本期视觉证据压缩成阵营的轮廓、权力物件与推荐主参考，原画可直接据此建立角色 brief。

F01 / NEW CARTEL

新派毒枭 · 湾流集团

企业化、跨境资产与低声谈判。看似能进入董事会，实际控制机库与航线。

轮廓
轻西装＋开放领＋低重心
权力物
航线册 / 机库卡 / 厚表
主参考
Landman / Griselda
F02 / OLD CARTEL

老派毒枭

地方关系、牧场权威与长期联盟；价值感来自“这里所有人都认识他”。

轮廓
亚麻衫＋短夹克＋靴
权力物
地图 / 旧表 / 家族照片
主参考
Narcos: Mexico / MobLand
F03 / WHITE PHARMA

白人药企

合法制度包裹地下利益；不做时尚精英，做能调动律师、安保和渠道的控制者。

轮廓
医学会议装＋克制大衣
权力物
样品箱 / 基金徽章 / 合同
主参考
The Gentlemen / The Outfit
F04 / MILITARIZED

军事化毒枭

战力藏在便装系统里；他是能部署小队的老板，不是把弹匣挂满胸口的士兵。

轮廓
硬肩便装＋局部软甲
权力物
加密机 / 旧军表 / 路线图
主参考
Gangs of London
F05 / MAFIA

黑手党

继承、座次、服务与家族纪律。西装只是制度的制服，不是单纯的昂贵标签。

轮廓
厚驳领＋长外套＋稳肩线
权力物
家族戒 / 会所钥匙 / 账本
主参考
MobLand / Peaky Blinders

飞车党

俱乐部历史比火焰纹更重要；磨损、章位和车辆改件必须能讲出资历。
